## Template Government Website

Proyek ini merupakan template **front-end website Organisasi Perangkat Daerah (OPD)**
yang dibuat menggunakan **HTML, CSS, dan JavaScript**. Template ini dirancang untuk membantu instansi 
pemerintah dalam membangun website resmi dengan tampilan yang modern, informatif, dan responsif dengan aksesn budaya jawa.

## Teknologi yang Digunakan
* **HTML5** – Struktur halaman website
* **CSS3** – Styling dan layout website
* **JavaScript** – Interaksi dan fungsi dinamis

## Struktur Folder Utama
* assets: Berisi semua aset statis yang digunakan pada website (css, images, pdf, video)
* Informasi: Berisi halaman informasi publik seperti berita, infografis, dan galeri.
* Layanan: Berisi halaman terkait layanan yang disediakan oleh OPD.
* PPID: Berisi halaman terkait keterbukaan informasi publik.
* Profile: Berisi halaman profil instansi, seperti tentang kami dan struktur organisasi.
* Survei: Berisi halaman survei kepuasan masyarakat dan FAQ.
* Ulas_Aduan: Berisi halaman untuk pengaduan masyarakat.
* home.html: Halaman utama website.
* script.js: Berisi JavaScript untuk interaksi website seperti navbar mobile, dropdown, animasi, dan tombol back to top.

## Struktur File CSS
* **base.css** → pengaturan dasar seperti reset CSS dan font
* **layout.css** → styling untuk layout utama seperti navbar, footer, dropdown, accessibility menu, dan animasi
* **component.css** → styling komponen umum seperti tombol, judul section, hero page, dan layout konten

## Cara Menjalankan Project
1. **Download atau Clone Repository**
   ```bash
   git clone https://github.com/kamilarr/Template_Government_Website.git
   ```
3. **Masuk ke folder project**
    ```bash
   Template_Government_Website
    ```
5. **Buka file utama**
   ```bash
   home.html
   ```
7. Website akan langsung berjalan di browser.
